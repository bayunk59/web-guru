Anda dapat mencarinya di dalam kode dengan menekan tombol Ctrl + F dan mengetik kata kunci MASUKKAN LINK.

Berikut adalah file kode lengkap yang telah dilengkapi komentar petunjuk pengisian link:

Saya telah menambahkan komentar petunjuk yang jelas di setiap baris dokumen. Untuk mengisinya, Anda hanya perlu:
